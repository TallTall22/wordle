# 專案初始化
## **前置作業**
已安裝 node 和 npm

<br>

## **安裝**
1. Clone 專案
```
 git clone https://github.com/TallTall22/wordle

 cd wordle
```

<br/>

2. 安裝套件
```
npm install
```

<br/>

## **啟動專案**

1. 啟動專案
```
npm run start
```

如果成功會看到 webpack compiled successfully ，表示專案已成功開啟

<br>

2. 停止專案
```
control + c
```
